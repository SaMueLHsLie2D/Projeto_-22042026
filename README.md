# Pedra Papel Tesoura

## Descrição
Pedra Papel Tesoura é um jogo clássico onde você joga contra a CPU. O objetivo é escolher entre Pedra, Papel ou Tesoura para vencer o oponente.

**Regras:**
- Pedra quebra Tesoura
- Tesoura corta Papel
- Papel cobre Pedra

## Características
- 🎮 Interface intuitiva e responsiva
- 🤖 Jogo contra a CPU com lógica aleatória
- 📊 Sistema de pontuação em tempo real
- 🎯 Feedback visual imediato dos resultados

## Tecnologias Utilizadas
- **HTML5** - Estrutura da página
- **CSS3** - Estilização e responsividade
- **JavaScript** - Lógica do jogo e interatividade

## Como Jogar
1. Escolha uma opção: **Pedra**, **Papel** ou **Tesoura**
2. A CPU fará sua escolha aleatoriamente
3. O resultado é mostrado instantaneamente
4. Sua pontuação é atualizada automaticamente
5. Jogue quantas vezes quiser!
